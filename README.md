# users-crud
Spring MVC + Hibernate RESTful CRUD

### Prerequisites

* JDK 8
* Maven
* PostgreSQL

### Getting Started

In root folder (where pom.xml is) run:
```
 mvn clean install tomcat7:run
```
Now, if there was no failures, tomcat server is running on [hhtp://localhost:8080](http://localhost:8080)
